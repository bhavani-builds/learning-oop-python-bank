# 🏦 Bank Account System (Python OOP)

A simple **Bank Account System** developed in Python using **Object-Oriented Programming (OOP)** concepts. This project demonstrates **Encapsulation** by using private attributes and methods to manage bank account operations securely.

---

## 📌 Features

- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- Uses private attributes for data security
- Beginner-friendly Python OOP project

---

## 🛠️ Technologies Used

- Python 3.x
- Object-Oriented Programming (OOP)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/bhavani-builds/python-bank-account-system.git
```

2. Navigate to the project folder

```bash
cd python-bank-account-system
```

3. Run the program

```bash
python bank_account.py
```
